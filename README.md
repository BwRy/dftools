# DFTools:

Convenient commandline scripts for working with dataframes in csv files.

## Install:

```bash
git clone git@github.com:gryan11/dftools.git
cd dftools
pip install .
```

## Usage:
```bash
showdf <csv-file>
statdf <csv-file>
cmpdf <csv-file1> <csv-file2>
latexdf <csv-file>
```

Run commands with `-h` for further usage options.
